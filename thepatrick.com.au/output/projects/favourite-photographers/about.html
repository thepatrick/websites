A relatively simple script that shows you whose photographs you Favourite most on Flickr, a relatively simple gauge of your favourite photographers.

Version 2 is purely JavaScript (with the help of a ruby-powered proxy to get around the limitations of XHR). Grab the source by looking at the source code of the main page and the fp class. The class is covered under my usual MIT license. I'll put up info (and the source) for the ruby proxy shortly (realistically it just protects my API key, and allows for shorter method names). If you want to know more just contact me.

More info on the original PHP version, including source is available in the My Favourite Photographers entry on Soapobx.
